# Pull Up / PU Project Instructions

Build Pull Up, also known as PU.

PU is a mobile-first live discovery app for college towns, starting with UIUC.

The app helps users answer:
"What’s the move tonight?"

## Product Identity

This is NOT a boring calendar app.

The emotional priority is:
1. FOMO
2. Belonging/social connection
3. Exploration/discovery

The app should feel:
- alive
- urgent
- social
- energetic
- local
- student-native

Think:
- Waze for campus activity
- TikTok-style discovery
- nightlife energy
- real-time social pulse

## Core Features

PU should display:
- frat parties
- bars/clubs
- student org events
- food deals
- campus events
- concerts
- sports watch parties
- pop-ups
- random local happenings

## Tech Stack

Use:
- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Zustand
- TanStack Query
- Supabase later
- Lucide icons

Do not use:
- Redux
- Firebase
- MongoDB
- GraphQL
- unnecessary complexity

## Design Rules

Default to dark mode.

The app should feel:
- premium
- modern
- fast
- mobile-first
- nightlife-inspired
- highly visual

Use:
- rounded cards
- urgency badges
- gradients
- smooth animations
- strong typography
- bottom navigation

Avoid:
- generic dashboards
- calendar-heavy layouts
- corporate SaaS design
- clutter

## MVP Screens

Build:
1. Tonight Feed
2. Event Detail Page
3. Map Screen
4. Deals Screen
5. Submit Event Screen
6. Admin Dashboard

## Event Card Requirements

Each event card should show:
- title
- category
- area/location
- time
- cover
- entry type
- stag rule
- age restriction
- vibe/music
- crowd status
- urgency labels
- image/flyer

Examples:
- Packed right now
- Free entry ending soon
- Trending tonight
- Line forming
- Ends in 30 mins

## Product Rules

Prioritize:
- emotional energy
- urgency
- social momentum

Do NOT build:
- messaging
- comments
- ticketing
- public attendee lists
- complex profiles

## Important

The Tonight Feed is the core product.

Everything should optimize for:
"What’s worth pulling up to right now?"
